# 📝 Todo List App

A simple and interactive Todo List application built using HTML, CSS, and JavaScript.  
This project started as a prompt-based application and was later upgraded to a UI-based web app with data persistence.

---

## 🚀 Features

- Add new tasks
- Delete existing tasks
- Clean and simple user interface
- Data stored using browser localStorage (persists after refresh)
- Beginner-friendly and easy to understand logic

---

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript (Vanilla JS)
- Browser LocalStorage

## ▶️ How to Run the Project

1. Download or clone the repository
2. Open the project folder
3. Open `index.html` in your browser
4. Start adding and managing your tasks

---

## 💡 Example Usage

- Enter a task in the input field
- Click **Add** to add the task
- Click **Delete** to remove a task
- Refresh the page → tasks will still be saved

---

## ⚠️ Limitations

- No edit/update feature
- No task prioritization
- Basic UI (can be improved further)
